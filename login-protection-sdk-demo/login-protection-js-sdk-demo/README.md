# 登陆保护 js sdk示例demo

js sdk示例demo，此demo会调用 login-protection-check-demo中的login.do接口

# demo运行步骤

* 修改login.html，替换sdk的配置

```

    <script type="text/javascript" 
        src="http://nos.netease.com/yidun/res-wm-1.0.js?t=0" 
        pro-data="YOUR_SDK_CONFIGURATION"></script>
        
```


* 将login.html复制到任意web容器中，并启动web容器

* 浏览器访问login.html
